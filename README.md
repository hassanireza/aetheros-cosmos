## ✦ Transmission Log

> *"BUILT FOR THE VOID."* AetherOS is an immersive, cinematic single-page experience that turns scroll position into spacecraft telemetry. Visitors drift from Earth's orbit, through the Sol Array, into deep space coverage and the observable universe — guided by full-bleed video fields, animated signal data, and a mission-control aesthetic.

## ✦ Quickstart

```bash
# ─── clone the array ───────────────────────────────
git clone https://github.com/<your-org>/aetheros-cosmos.git
cd aetheros-cosmos

# ─── install dependencies ──────────────────────────
npm install

# ─── boot local observatory (dev) ──────────────────
npm start
# → http://localhost:3000

# ─── compile production build ──────────────────────
npm run build
# → outputs to /build, ready for static hosting
```
